# Udemy Clone – Frontend Project

## Project Overview

This project is a static front-end clone of the Udemy homepage developed using HTML5 and CSS3. The goal of this project was to replicate the layout and structure of a professional e-learning platform while strengthening my understanding of web development fundamentals.

The application consists of two main pages: a Home page and a Contact page. The focus of the project is entirely on user interface design and layout implementation.

## Pages Included

Home Page  
The Home page includes a structured navigation bar, course categories section, featured courses display, course cards with details such as title, instructor name, rating, and price, and a footer section.

Contact Page  
The Contact page contains a basic contact form layout designed using HTML and styled with CSS.

## Technologies Used

HTML5  
CSS3  
Flexbox for layout design

## Objective

The primary objective of this project was to practice semantic HTML structuring, improve CSS styling and layout techniques, and understand how real-world websites are visually organized.
